스타벅스의 Siren order와 같이, 손님, 매장 간의 메뉴를 주문 및 처리가 가능한 시스템

## 기능

- 손님 메뉴 주문
- 각 매장의 위치 확인
- 메뉴 재고 관리
- 실시간 주문 처리

## 사용 기술 스택

- react
- node Express
- JWT
- socket.io
- sequelize
- google Maps API

### `npm install` : 프로젝트에 필요로 한 라이브러리 설치

### `npm start` : 서버 실행
